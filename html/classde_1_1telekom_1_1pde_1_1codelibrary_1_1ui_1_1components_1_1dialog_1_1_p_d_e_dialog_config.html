Dialog configuration data class. More...

Inherits Parcelable.

Detailed Description

Dialog configuration data class.

This class stores all data that is needed to configure a constructDialog. Since the actual visual representation of the dialog is wrapped into a separate activity this class is parcelable in order to deliver an easy way to pass the configuration data (encapsulated in an object) along by an intent.
