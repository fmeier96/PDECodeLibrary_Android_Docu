Detailed Description

The PDESectionedButton is a simple button type with 1 or more sections layouted horizontally next to each other. Internally it works with the normal PDEButton, so the events are the same as the PDEButton has, the only difference at the moment is that the PDESectionedButtonEvent has additional information about the index of the button which was clicked.

You can dynamically insert or set each section by hand in the code, or use a xml array with predefined sections.

XML information:

At first you have do define every sections with 4 items

  1. Text in the section -> String (With or without ") or string reference 2. Icon in the same section -> drawable reference, string path (With or without ") , iconstring (z.b "#p"; need " in this case) ## default is null
  2. Colored -> boolean reference, false, true, 0, 1 ## default is false
  3. Enabled -> boolean reference, false, true, 0, 1 ## default is true

If one information is not needed (because you want the default value) , you can keep it empty "<item></item>", or remove it completely from the array if the following items are also not needed because they are the same as the default values.

!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!! !!! You can't have different order -> Icon,colored,enabled,text !!! !!!! If you do this you will have unexpected behaviour!!!!!!!!!!!! !!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!

Valid Examples: <array name="test"> First //Text icon.png //icon true // icon colored </array>

<array name="test"> First // Text // no icon // not colored -> default false // disabled </array>

<array name="test"> First // Text </array>

<array name="test"> First "#p" </array>

XML Example:

<resources> <array name="SectionedButton_sample_default_section_1"> First </array> <array name="SectionedButton_sample_default_section_2"> Second </array> <array name="SectionedButton_sample_default_section_3"> Third </array>

<array name="SectionedButton_sample_default"> /SectionedButton_sample_default_section_1 /SectionedButton_sample_default_section_2 /SectionedButton_sample_default_section_3 </array> </resources>
