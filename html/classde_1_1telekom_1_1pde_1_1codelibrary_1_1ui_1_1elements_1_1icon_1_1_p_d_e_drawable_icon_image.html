void de.telekom.pde.codelibrary.ui.elements.icon.PDEDrawableIconImage.updateAllPaints ( )
inlineprotectedvirtual

Update all used Paint-Instances here.

If global paint properties like e.g. alpha, dither, colorFilter, etc. change all important Paint-Instances that are used within this drawable have to be updated. Place the code to recreate the Paints (with the new values) within this method. You can also use it for updates of a distinct paint (e.g. when a distinct color has changed), although this means some overhead. This method HAS to be overridden by derived classes.
