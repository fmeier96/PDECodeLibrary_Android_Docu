Detailed Description

PDEList example with multiple lines of text.

In this example we have a main text and a subtext. So we have at least two lines of text. The subtext can be switched to have more than one line of text.

Helper that creates the whole data set with the configured number of items.

There are also two layout variants. In one of them there is just one line of content text, in the other one there are multiple lines of content text. The amount of content text which is stored within the data set depends on the chosen layout. So we produce two data sets with this method. The decision which one is produced is regulated by the parameter

Parameters
numberOfLinesDecides for which layout the data set is produced.
