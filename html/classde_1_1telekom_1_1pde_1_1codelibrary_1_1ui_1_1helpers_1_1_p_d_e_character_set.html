Character set with int number digits (contains signs).

When reading integer numbers by using the character set, be aware that malformed integers (e.g. 12+-3) might also be returned. These integers might not be converted correctly to into numbers.

When reading float numbers by using the character set, be aware that malformed floats (e.g. 1.3.0ee+-17) might also be returned. These floats might not be converted correctly to into numbers.
