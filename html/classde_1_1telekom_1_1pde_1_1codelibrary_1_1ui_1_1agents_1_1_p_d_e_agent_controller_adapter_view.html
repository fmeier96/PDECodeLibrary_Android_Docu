Detailed Description

Adapter for button style UI elements to AgentController.

This class links an AgentController to a UIControl. It consumes the button's events and passes them on to the AgentController in a suitable form.

Causes the Runnable r to be added to the message queue, to be run after the specified amount of time elapses.

Convenience function. We use this mechanism in order to start delayed the runnable which checks if we're still in touchdown-mode.

Parameters
rThe Runnable that will be executed.
delayMillisThe delay (in milliseconds) until the Runnable will be executed.
Returns
Returns true if the Runnable was successfully placed in to the message queue.
