Detailed Description

Event state sent along with all slide events.

Contains the complete state of the slider changes. Each value is given in the 0..1 range that the slider content uses for setting up the slider and it's equivalent out of the user defined value range. The slider contents should extract the necessary values for themselves.

Sliders can contain different controllers, for assigning them, a event comes with an id.
