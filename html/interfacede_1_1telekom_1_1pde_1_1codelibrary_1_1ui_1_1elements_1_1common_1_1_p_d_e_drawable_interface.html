int de.telekom.pde.codelibrary.ui.elements.common.PDEDrawableInterface.getNeededPadding ( )

Returns the padding the element needs to be displayed correctly.

Some things like an outer shadow have to be drawn outside of the element bounds. So the View that holds the element has to be sized bigger than the element bounds. For proper layouting the view must be extended to each direction by the value delivered by this function.

Returns
the needed padding
