float de.telekom.pde.codelibrary.ui.elements.common.PDEDrawableShapedShadow.getMaxBlurRadius ( )
inline

Get maximum blur radius.

Delivers the maximum blur radius that was set in the lifetime of this object. We need the maximum blur radius for the calculation of a proper dirty rect. It's not the most efficient solution, since the dirty rect will in many cases be bigger than it would have to be, but it prevents drawing errors. Maybe we find a better solution later on.

float de.telekom.pde.codelibrary.ui.elements.common.PDEDrawableShapedShadow.getBiggerBlurRadius ( )
inline

Compares current and last blur radius and delivers bigger one.

The blur radius can grow or shrink. For a quite optimal dirty rect calculation we need to compare the current radius with the last one and use the bigger one of both.

inline

Set the path to use.

Paths are always filled using the count rule (which is an iOS feature and cannot be changed at the moment). So something like shapes with holes etc. cannot be done through this layer.
