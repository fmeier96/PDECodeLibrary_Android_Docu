Detailed Description

Handles OAuth authentication with OneIDM Server.

The class implements two methods to request an access token from the server. It is used by PDEOneIDMLoginScreenActivity - here the requestAccessToken with username and password is used. In order to get a new access token if you already have a refresh token you use the requestAccessToken function which take the refresh token as parameter.
