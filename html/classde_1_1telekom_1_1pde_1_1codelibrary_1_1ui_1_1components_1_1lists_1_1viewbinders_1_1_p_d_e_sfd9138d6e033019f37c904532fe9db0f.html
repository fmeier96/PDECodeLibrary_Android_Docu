Detailed Description

Custom View Binder which enables the use of various PDE components for a SimpleCursorAdapter.

A standard Android SimpleCursorAdapter can only handle some default Android UI components. But the SimpleCursorAdapter offers the possibility to use custom ViewBinders. With this ViewBinder interface it is possible to add further UI components and to determine how they will be filled with data. So this ViewBinder handles PDE UI components which are commonly used within list element layouts.

Recommendation: The PDESimpleCursorAdapter automatically makes use of this ViewBinder. It comes along with some list element highlighting logic in the style of the styleguide. So we recommend to simply use the PDESimpleCursorAdapter.
